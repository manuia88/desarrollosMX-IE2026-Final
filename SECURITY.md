# Security Policy

## Reporting a Vulnerability

Report suspected security vulnerabilities to **security@desarrollosmx.com**.

We acknowledge receipt within 72 hours and commit to a public disclosure window of up to **90 days** from the initial report while we investigate and ship a fix. Early disclosure may occur when a fix is available sooner.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Affected components, versions, or URLs
- Optional: suggested remediation

Do **not** open public GitHub issues for security reports.
