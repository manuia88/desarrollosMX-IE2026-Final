Middleware base (FASE 02):
- `publicProcedure`: sin auth.
- `authenticatedProcedure`: requiere `ctx.session.user`. Rate limited 600/min.
- `asesorProcedure`: authenticatedProcedure + `ctx.profile.rol in ('asesor','master_broker','admin','superadmin')`. Rate limited 300/min.
- `developerProcedure`: rol `desarrollador` o `admin`.
- `contadorProcedure`: rol `contador` o feature `contabilidad_access` override.
- `compradorProcedure`: rol `comprador`.
- `adminProcedure`: rol `admin` o `superadmin` + logs en `admin_actions`.
- `featureProcedure(key)`: verifica `check_feature_limit(userId, key)`.

Middleware nuevos (H2/H3 — biblia v2 moonshot):
- `partnerProcedure`: rol `partner` (API B2B bilateral contracts, FASE 33). Rate limited por tier contractual (silver 100/min, gold 1K/min, platinum 10K/min). Valida `partner_api_keys.active=true` + `contract_status='active'`.
- `kycVerifiedProcedure`: requiere `profiles.kyc_status='verified'` + nivel ≥ 2. Gate específico para fractional investing (FASE 36). Integra con proveedor KYC (Mati / Truora en LATAM). 
- `chromeExtProcedure`: valida requests HMAC-signed desde Chrome extension DMX (GC-27 / ADR-012). Header `X-DMX-Ext-Signature: hmac-sha256(body, shared_secret)` + `X-DMX-Ext-Version`. Rate limit 500 captures/hora/usuario. Blocklist de hostnames server-side (no se auto-suscribe).

Convención: todo `input` es Zod schema nombrado `<procedure>Input`. Toda `mutation` sensible crea row en `audit_log` (ADR-018 Regla 7 — E2E Connectedness, cumplimiento obligatorio).

## Middleware stack detalle

```ts
// server/trpc/middleware.ts
export const rateLimited = t.middleware(async ({ ctx, path, next }) => {
  const key = ctx.session?.user?.id ?? ctx.ip;
  const ok = await checkRateLimitDb(key, path, 300);
  if (!ok) throw new TRPCError({ code: 'TOO_MANY_REQUESTS' });
  return next();
});

export const authenticated = t.middleware(({ ctx, next }) => {
  if (!ctx.session?.user) throw new TRPCError({ code: 'UNAUTHORIZED' });
  return next({ ctx: { ...ctx, userId: ctx.session.user.id }});
});

export const requireRole = (roles: string[]) => t.middleware(({ ctx, next }) => {
  if (!roles.includes(ctx.profile.rol)) throw new TRPCError({ code: 'FORBIDDEN' });
  return next();
});

export const requireFeature = (key: string) => t.middleware(async ({ ctx, next }) => {
  const limit = await checkFeatureLimit(ctx.userId, key);
  if (!limit.enabled) throw new TRPCError({ code: 'FORBIDDEN', message: `feature_gated:${key}` });
  if (limit.remaining !== null && limit.remaining <= 0) throw new TRPCError({ code: 'PAYMENT_REQUIRED', message: `limit_exceeded:${key}` });
  return next({ ctx: { ...ctx, featureLimit: limit } });
});

// Middlewares añadidos en biblia v2 moonshot (H2/H3)

export const chromeExtProcedure = authenticated
  .use(async ({ ctx, next, rawInput }) => {
    const sig = ctx.req.headers['x-dmx-ext-signature'];
    const version = ctx.req.headers['x-dmx-ext-version'];
    const expected = hmacSha256(JSON.stringify(rawInput), process.env.DMX_EXT_SHARED_SECRET!);
    if (sig !== expected) throw new TRPCError({ code: 'UNAUTHORIZED', message: 'invalid_extension_signature' });
    if (!isSupportedVersion(version)) throw new TRPCError({ code: 'BAD_REQUEST', message: 'extension_outdated' });
    return next({ ctx: { ...ctx, extVersion: version } });
  })
  .use(rateLimited(500, 'hour'));

export const partnerProcedure = t.procedure
  .use(async ({ ctx, next }) => {
    const apiKey = ctx.req.headers['x-partner-api-key'];
    if (!apiKey) throw new TRPCError({ code: 'UNAUTHORIZED' });
    const partner = await loadPartnerFromKey(apiKey);
    if (!partner.contract_active) throw new TRPCError({ code: 'FORBIDDEN', message: 'contract_inactive' });
    const tierLimits = { silver: 100, gold: 1000, platinum: 10000 };
    await checkRateLimitDb(partner.id, ctx.path, tierLimits[partner.tier]);
    return next({ ctx: { ...ctx, partner } });
  });

export const kycVerifiedProcedure = authenticated
  .use(async ({ ctx, next }) => {
    const kyc = await getKycStatus(ctx.userId);
    if (kyc.status !== 'verified') throw new TRPCError({ code: 'FORBIDDEN', message: `kyc_required:${kyc.status}` });
    if (kyc.level < 2) throw new TRPCError({ code: 'FORBIDDEN', message: 'kyc_level_insufficient' });
    return next({ ctx: { ...ctx, kyc } });
  });
```
