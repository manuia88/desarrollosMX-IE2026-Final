# 08.3 — Product Catalog (empaquetados por persona)

> **Propósito:** catálogo de ~18 productos DMX empaquetados organizados por persona objetivo (ver 08.2 PERSONA_MAP). Cada producto combina features del FEATURE_INVENTORY (08.1) en paquete cohesivo con pricing, time-to-value, diferencial competitivo.

# 🏗️ Productos Developer (4 productos)

## Producto 1.1 — "DMX LandAnalyzer"
> **JTBD:** "Tengo terreno y necesito saber qué construir ahí"

**Features bundle:**
- S-B06 Project Genesis + S-H13 Site Selection AI (N3)
- S-B15 Launch Timing (N2) + S-I01 AVM
- S-DMX-IPV Precio-Valor index
- L-60 LandAnalyzer Wizard UI
- L-71 Branded PDF Report output
- CF-L3 "LandAnalyzer Enterprise" cross-function

**Input del usuario:** lat/lng terreno + presupuesto + timeline + vocación preferida
**Output:** PDF report con: tipo proyecto recomendado (residencial/mixed/comercial), mix unidades óptimo, pricing esperado, absorción proyectada, timing lanzamiento, ROI 5 años, riesgo score + mitigaciones

**Tiers pricing:**
- **Pro**: $1,999 USD/mes (5 análisis/mes incluidos)
- **Enterprise**: $50K MXN/año (20 análisis/mes + dedicated support)
- **One-off**: $3K-$8K USD por análisis (para no-suscriptores)

**Time-to-value:** 24-48h desde upload terreno hasta report completo
**Competidor:** Brandata (estudio vocación manual consulting) + DatAlpine Enterprise Package
**Diferencial DMX:** Automated + Monte Carlo scenarios + 118 scores IE jerárquicos + multi-país LATAM
